# Blogicum - Платформа для ведения блогов


## О проекте
**Автор** - Шерокожухов Дмитрий Денисович.

**Связь** - Почта автора: cherokoko255@yandex.ru

**Blogicum** - это современная веб-платформа для создания и ведения блогов. Проект разработан на фреймворке Django и предоставляет пользователям полный набор инструментов для публикации контента, взаимодействия с другими пользователями через комментарии и управления своим профилем.

### Основные возможности

| Функция | Описание |
|---------|----------|
| Регистрация и авторизация | Создание аккаунта, вход, восстановление пароля |
| Управление публикациями | Создание, редактирование, удаление постов |
| Комментарии | Добавление, редактирование, удаление комментариев |
| Категории и локации | Группировка постов по темам и местам |
| Изображения | Загрузка изображений к публикациям |
| Отложенные посты | Планирование даты публикации |
| Пагинация | Постраничный вывод постов (10 на страницу) |
| Профиль пользователя | Просмотр и редактирование личной информации |

## Технологический стек

- **Python** 3.10 - язык программирования
- **Django** 3.2.16 - веб-фреймворк
- **SQLite3** - база данных
- **Bootstrap 5** - CSS фреймворк
- **Pillow** - обработка изображений

## Установка и запуск

### Требования

- Python 3.10 или выше
- pip (менеджер пакетов)
- Git

### Инструкция по установке

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/izikatkaWN/django_sprint4.git
cd blogicum

# 2. Создайте виртуальное окружение
python -m venv venv

# 3. Активируйте виртуальное окружение
# Для Windows:
venv\Scripts\activate
# Для Linux/Mac:
source venv/bin/activate

# 4. Установите зависимости
pip install -r requirements.txt

# 5. Перейдите в директорию проекта
cd blogicum

# 6. Выполните миграции базы данных
python manage.py migrate

# 7. Загрузите тестовые данные (опционально)
python manage.py loaddata db.json

# 8. Создайте суперпользователя (администратора)
python manage.py createsuperuser

# 9. Запустите сервер разработки
python manage.py runserver
